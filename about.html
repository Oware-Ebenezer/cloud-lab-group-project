Project Overview

This static website is created as part of our Cloud Computing Practical Lab 1. The goal is to demonstrate our understanding of virtualization concepts and static website deployment using GitHub Pages.

Through this project, we are learning how to:

  • Collaborate using Git and GitHub
  • Build a multi-page static website with HTML, CSS, and JavaScript
  • Deploy a live website using GitHub Pages
  • Work effectively as a team with assigned roles

Virtualization Context

This project complements our hands-on work with Oracle VM VirtualBox, where we learned to create and manage virtual machines. Just as virtualization allows multiple operating systems to run on one physical machine, GitHub Pages allows us to host multiple static websites from a single repository.
